# 🐧 Linux Administration Notes

A beginner-friendly Linux Administration notes repository created while learning Linux for System Administration, Technical Support, Cloud Computing, and DevOps.

This repository contains Linux concepts, commands, practical exercises, interview questions, and screenshots of command outputs.

---

## 📚 Topics Covered

- Users
- Groups
- Permissions
- Ownership (chown)
- Services (systemctl)
- Cron Jobs
- SSH
- System Logs
- Disk Management
- File Searching (find, grep)
- Archiving (tar)
- File Synchronization (rsync)

## 🎯 Purpose

The purpose of this repository is to:

- Learn Linux Administration from beginner to intermediate level.
- Practice Linux commands with real examples.
- Build a GitHub portfolio for Technical Support, System Administrator, Cloud, and DevOps roles.
- Prepare for Linux interview questions.

---

## 🛠️ Practice

Run all commands on:

- Ubuntu
- WSL (Windows Subsystem for Linux)
- Virtual Machine (VirtualBox/VMware)
- Linux Server

Take screenshots after executing commands and store them inside the `screenshots/` folder.

## 📖 Learning Outcome

After completing this repository, you will be able to:

- Manage Linux users and groups.
- Configure file permissions and ownership.
- Manage Linux services.
- Schedule tasks using Cron.
- Access remote systems using SSH.
- Analyze system logs.
- Manage disks and storage.
- Search files efficiently.
- Create and extract archives.
- Synchronize files using rsync.
